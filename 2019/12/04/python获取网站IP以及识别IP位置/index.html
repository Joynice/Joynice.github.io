获取IP所在地理位置

刚才的需求解决了，那么得到IP，我就想获得IP地址所在的位置如：国家、城市等等，本想这个功能应该只能调用第三方接口了吧，但是谷歌一下，发现python中有实现这个功能的库geoip2，简单的浏览下它的官网，发现他们实现这个功能就是基于数据库比对，可以实现以下功能：

  • IP位置查询
  • IP是否高匿，解析真实地址
  • 域名查询
  • 连接方式查询
  • ASD查询
  • ISP查询
